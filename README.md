# Local Uploads
Tools & Assets:
- HTML, CSS, Javascript
- Bootstrap, Font Awesome
- PHP

## How to run
- Insert the project folder in the directory that Apache is hosting. In case you are using XAMPP it's usually *"C:/xampp/htdocs/"*.
- Start in index.php
